dicas de como usar o terminal

alguns exemplos de como podemos utilizar o terminal para otimizar nossa performance e melhorar o fluxo de estudos e trabalho :)

comece com uma pasta vazia

crie uma pasta vazia na sua área de trabalho, documentos ou uma pasta específica para estudos. chame ela de "terminal-teste"

abra o terminal no vscode

abra a pasta "terminal-teste" no seu vscode e abra o terminal clicando em "terminal" ou segunrando e arrastando para cima a barra inferior

crie um documento

para criar um novo documento nessa pasta vazia, precisa usar o comando "touch" + "nome do arquivo.formato do arquivo", exemplo:

touch index.html

crie uma nova pasta

se você deseja criar uma nova pasta dentro da pasta "terminal-teste", utilize o comando "mkdir" + "nome-da-pasta", exemplo:

mkdir assets

descobra seu local atual

você não precisa lembrar em qual pasta você está atualmente, para descobrir basta fazer o comando "pwd"

pwd

navegue entre pastas

para criar um arquivo ou pasta dentro da nova pasta chamada "assets", você precisa navegar até essa pasta, se você criar onde está, vai criar na pasta "terminal-teste". para entrar na pasta "assets" voce deve usar o comando "cd" + "nome-da-pasta", exemplo:

cd assets

retornar para a pasta "terminal-teste" basta usar "cd" + "..", exemplo:

cd ..

saiba o que existe dentro da pasta

para descobrir o que existe de documento ou pasta dentro da qual você está, basta usar o comando "ls" para listar

ls

sobre o git

o comando "git init" inicia o seu repositório no git, é importante usar esse comando no inicio do seu projeto novo. o comando "git status" mostra o status do seu repositório, se existem modificações no seus arquivos e pastas, adições e remoções. ali também diz se já criou o ramo main e se houve commits. ele sugere adicionar os arquivos com o comando "git add"

para saber mais sobre git: curso sobre git e github gratuito
